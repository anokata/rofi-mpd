# mpd-menu

## Description
Simple mpd client in rofi. 

### Usage
Use `--help` or `-h` flag to get help.
```
usage: rofi-mpd [-h] [-l] [-s] [-a]

arguments:
-h, --help      show this message and exit
-l              library mode (artist -> album)
-s              song mode (select one song)
-a              ask for mode
```
